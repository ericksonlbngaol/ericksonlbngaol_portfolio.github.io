In this intern project, my team and I built an e-government application for the Regional Government of North Tapanuli Regency, based on a website. This application is developed with PHP programming (Bootstrap YII2 Framework) with SQL database. The application functions as a local government program planning application, keeping Musrenbang (Development Planning Consultation) and others, as a supporting local government program.
